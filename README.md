# Drive Wines GitHub Pages site

A lightweight static homepage inspired by the current Drive Wines site.

## Publish on GitHub Pages
1. Create a new public GitHub repository, such as `drive-wines-site`.
2. Upload everything in this folder, keeping the `assets` folder intact.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. GitHub will provide a temporary `github.io` address.

## Connect drivewines.com
In GitHub Pages settings, enter `www.drivewines.com` under **Custom domain**. GitHub will show the DNS records to add at your domain provider. Keep the old site active until the GitHub preview is approved and the DNS records are ready to change.

## Important before launch
- Replace the email-based signup form with your current newsletter provider’s embed form.
- Verify the shop, club, social, policy and account links.
- The two About/Vineyard background images currently load from the existing WordPress site. Download those into `assets` and change their URLs in `styles.css` before cancelling the old hosting.
- Confirm you have permission to reuse all included photographs and product images.
